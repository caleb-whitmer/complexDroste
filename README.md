# complexDroste

The complexDroste repository is an experimental project which edits the logarithmic space of Droste images to produce some cool results. The Droste module provides for the creation of both static images as well as animations and comes with a variety of functions which can be applied to the logarithmic space of an input. Checkout the [`gallery/`](https://github.com/caleb-whitmer/complexDroste/tree/main/gallery) directory to see some of the results!

<ins>This project was inspired by 3Blue1Brown's video which can be found</ins> [here](https://www.youtube.com/watch?v=ldxFjLJ3rVY).

## Build Instructions

Run `stack build` to build the project, and `stack run` to execute.

The [`app/Main.hs`](https://github.com/caleb-whitmer/complexDroste/blob/main/app/Main.hs) can be edited to make new images using the Droste module.
